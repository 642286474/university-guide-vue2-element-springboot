# university-guide-vue2-element-springboot

## 简介

​		基于Vue2 + SpringBoot的大学漫游指南。创意源于百度地图、高德地图等成熟的商业地图在校园内信息服务方面的不满足，由于这些成熟的商业地图所涉及的是世界范围内的所有地图信息，因此更多的是专注于提供地理位置的抽象信息服务，而大学漫游指南专注的则是为大学生提供大学内的信息指引服务。
